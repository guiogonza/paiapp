# ⚠️ INSTRUCCIONES PARA AGREGAR EL LOGO PPAI

## Paso 1: Coloca tu imagen
Coloca tu archivo de imagen del logo PPAI en esta carpeta con el nombre exacto:

**`ppai_logo.png`**

## Paso 2: Verifica la ubicación
El archivo debe estar en:
```
assets/images/ppai_logo.png
```

## Paso 3: Reinicia la aplicación
Después de colocar la imagen, necesitas hacer un **HOT RESTART** (no solo hot reload):

1. Detén la aplicación (Ctrl+C o Cmd+C)
2. Ejecuta: `flutter run -d chrome`
3. O en VS Code/Cursor: presiona `Ctrl+Shift+F5` (o `Cmd+Shift+F5` en Mac)

## ✅ Verificación
Si la imagen se carga correctamente, verás el logo PPAI en:
- Pantalla de Login (con fondo blanco)
- Pantalla de Splash (con fondo azul)

Si ves un ícono de "imagen no soportada", significa que:
- La imagen no está en la ubicación correcta
- El nombre del archivo no es exactamente `ppai_logo.png`
- Necesitas hacer un HOT RESTART (no solo hot reload)
